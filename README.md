# 🏦 Mi Banka - Dashboard de Finanzas Premium
---

## ✨ Descripción

**Mi Banka** es una plataforma de gestión financiera personal y empresarial diseñada con una interfaz moderna, responsiva y de alto impacto visual. Permite centralizar el control de múltiples cuentas bancarias, negocios y categorías, ofreciendo una vista unificada de todos los movimientos y transferencias en tiempo real.

Este proyecto utiliza un diseño de **Glassmorphism** con efectos **Neón**, optimizado para una experiencia fluida tanto en escritorio como en dispositivos móviles y tablets.

## ✨ Vista Previa del Dashboard

El sistema cuenta con un panel principal que muestra:
- 💳 **Resumen por Banco**: Tarjetas dinámicas con balances calculados automáticamente.
- 📊 **Balance Total**: Sumatoria global de ingresos vs gastos.
- 🔄 **Acciones Rápidas**: Accesos directos para nuevas transacciones y transferencias.

---

## 🔥 Características Principales

- 📱 **Diseño 100% Responsivo**: Tablas inteligentes que se transforman en tarjetas en dispositivos móviles.
- 📑 **Vista Unificada de Actividad**: Consulta en una sola lista tanto los movimientos (ingresos/gastos) como las transferencias internas de cada cuenta.
- 🏢 **Soporte Multinegocio**: Clasifica tus finanzas por diferentes unidades de negocio.
- 🏷️ **Categorización Inteligente**: Organiza tus flujos de dinero por categorías personalizables.
- 🛡️ **Seguridad y Estabilidad**: Implementación de sentencias preparadas (PDO) y manejo de errores robusto.
- 🕒 **Historial por Meses**: Organización automática de registros por períodos mensuales para un mejor análisis.

## 🛠️ Tecnologías Utilizadas

[![My Skills](https://skillicons.dev/icons?i=html,css,php,mysql,github,vscode,windows,&theme=light&perline=8)](https://skillicons.dev)

- **Backend**: PHP 8.x con PDO para una conexión segura a base de datos.
- **Base de Datos**: MySQL / MariaDB.
- **Frontend**: Vanilla CSS (Efectos Neón & Glassmorphism) y JavaScript puro.
- **Iconografía**: [Material Design Icons (MDI)](https://materialdesignicons.com/).
- **API**: Geocodificación para registro de visitas (opcional).

## 🚀 Instalación y Configuración

1. **Requisitos Previos**: Tener instalado un servidor local como **XAMPP**, **WampServer** o **Laragon**.
2. **Base de Datos**:
   - Crea una base de datos en `phpMyAdmin`.
   - Importa el archivo SQL proporcionado (ej. `mi_banka.sql`).
   - Configura tus credenciales en `conexion.php`.
3. **Despliegue**:
   - Clona este repositorio o copia los archivos en la carpeta `htdocs`.
   - Accede desde tu navegador a `http://localhost/tu-carpeta/mi_banka/`.
